# 中学校向け時間割作成ツール - ソースコード解説

> 非エンジニア向け（学校の先生や管理職向け）のソースコード解説ドキュメント

## このアプリについて

このツールは、**中学校の時間割を効率的に作成・管理するために作られたアプリケーション** です。先生が授業の割り当てをドラッグ&ドロップで操作でき、制約条件（例：先生の担当時数、連続授業の制限など）を自動チェックしながら、最適な時間割を作ることができます。

### アプリの3つの起動方法

1. **ブラウザ版** - インターネットブラウザで開いて使う
2. **デスクトップ版（Windows）** - `.exe`ファイルとして起動する（Electron）
3. **開発環境** - コードを編集しながら実行する（Vite）
