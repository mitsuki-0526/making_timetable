# TT設定タブ

> 対応ソースコード: `src/components/settings-tabs/TtAssignmentsTab.tsx`

## このファイルの役割

TT（チームティーチング）の授業ルールを登録する画面です。

ここでは「何の教科群を」「何年の学年群で」「どのクラス群で」「誰が一緒に担当するか」をまとめて設定できます。

---

## 主な機能

### TT設定の追加

- 設定名
- 教科（複数選択可）
- 学年（複数選択可）
- 対象クラス（学年ごとに見出し付きで複数選択可）
- 参加教員（複数選択可）
- 有効/無効

これらを入力して、TT 設定を新しく作成できます。

- 学年を選ぶと、その学年ごとに「1年」「2年」の見出しが出ます
- 各見出しの下に、その学年で使える組のボタンが並びます
- これにより「1年1組だけ」「2年3組だけ」のように、学年と組をセットで指定できます

### TT設定の編集・削除

- すでに登録した TT 設定は一覧で表示されます
- 「編集」で内容を変更できます
- 「削除」で不要な設定を消せます

### 保存と読込への反映

- 登録した TT 設定は時間割データの JSON に含まれます
- 読み込み時も復元されます

---

## 現在の範囲

この画面で登録した TT 設定は、次の場所まで反映されます。

- 教科を手で入れた時の自動 TT 反映
- solver の自動配置
- 左パレットの TT タブ
- 保存・読込

左パレットの TT タブでは、この画面で作った設定をドラッグしてセルへ適用できます。

- 教科が1つだけの TT 設定なら、空セルへドロップした時に教科ごと設定できます
- 教科が複数ある TT 設定なら、すでに一致する教科が入っているセルにだけ適用します
