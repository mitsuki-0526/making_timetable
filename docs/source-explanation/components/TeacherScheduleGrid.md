# 先生のスケジュール表

> 対応ソースコード: `src/components/TeacherScheduleGrid.jsx`

## このファイルの役割

各先生がどの時間に何のクラスの授業を持っているかを、先生ごとに一覧で表示するコンポーネントです。学校の職員室に貼ってある「先生用の時間割表」をWebアプリで再現している、と想像してください。

---

## 主な機能

### 先生ごとのスケジュール表示

- **何をするか**: 
  - 縦軸：学校の全先生のリスト
  - 横軸：曜日（月火水木金）と時間（1限～6限）
  - セル内容：そこに授業が入っていれば「〇年〇組」と「教科名」を表示
- **背景色で区別**：
  - 薄い青：その先生が単独で担当
  - 薄い紫：B週の隔週授業
  - 薄い緑：先生グループで担当
  - 薄い黄：複数クラスの合同授業
- **ソースコード**: 行7-225（全体のコンポーネント）

### 週のコマ数の合計表示

- **何をするか**: 各先生の右側に「週計」という欄があり、その先生が週全体で何コマ授業を持っているか表示
  - 例：「田中先生 18コマ」
- **使い道**: 先生の労働時間が適切に配分されているか確認する
- **ソースコード**: 行53-65（週コマ数のカウント関数）、行130-137（表示部分）

### セル内容の詳細表示

#### 通常授業（単独担当）

- **表示内容**: 
  - クラス名（例：1-A）
  - 教科名
  - 色：薄い青
- **見た目のコード**: 行156-165（背景色と文字色の設定）

#### B週隔週授業

- **表示内容**:
  - クラス名
  - 教科名
  - 「B週」というバッジ（小さなタグ）
  - 色：薄い紫
- **背景**: A週とB週で違う授業を持つ先生の場合、B週の方が紫色で表示されます

#### 先生グループで担当

- **表示内容**:
  - クラス名
  - 教科名
  - 「👥」マーク
  - 色：薄い緑
- **背景**: 複数の先生で一緒に担当する授業（体育で複数の体育の先生で分担など）

#### 複数クラスの合同授業

- **表示内容**:
  - 複数クラスの名前が上下に並ぶ
  - 教科名
  - 「🔗合同」バッジ
  - 色：薄い黄
- **背景**: 複数のクラスが一緒に授業する場合（例：1年全学年で一緒に朝礼）
- **ソースコード**: 行179-192（合同授業の表示）

### データの正確性

- **グループ対応**: 先生グループに属している先生も、正しく認識されます
  - つまり「チームティーチング」で複数の先生が入るクラスも、全員の個別スケジュールに正しく表示されます
- **隔週対応**: A週とB週で別の先生が担当する場合も正しく表示されます

## 背景色の意味

| 色 | 意味 |
|---|---|
| 薄い青 | 単独担当（通常） |
| 薄い紫 | B週（隔週授業） |
| 薄い緑 | 先生グループで担当 |
| 薄い黄 | 複数クラスの合同授業 |
| グレー | その時間に授業がない（－で表示） |

---

## 実装のポイント

### getEntries 関数

このコンポーネントの心臓部分で、以下を判定します：
- 先生が直接担当しているか
- B週の隔週授業として担当しているか
- 先生グループ経由で担当しているか
- 複数クラスの合同授業か

**ソースコード**: 行11-37

### 複数クラス合同時の表示

複数クラスが一緒に授業する場合、1つのセルに複数のクラス名が上下に並びます。
- 複数クラスは1コマとしてカウント（重複カウントしない）
- ただし先生の負担は正確に計算される

---

## データの流れ

1. 時間割表（TimetableGrid）で授業が配置される
2. その情報がデータストアに保存される
3. TeacherScheduleGrid が データストアから情報を読み込む
4. 各先生ごとに「この時間に授業があるか」を確認
5. あれば、内容を表示します
