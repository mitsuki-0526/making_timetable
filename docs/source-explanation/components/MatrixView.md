# 全校時間割マトリクス

> 対応ソースコード: `src/components/MatrixView.tsx`

## このファイルの役割

このファイルは、学校全体の時間割を 1 枚の大きな表で見渡すための画面です。

- 左側: クラス名
- 上側: 曜日と時限
- 中央: 各クラスの授業内容

学年ごとの週表示とは違い、複数クラスをまとめて見ながら空きコマや配置の偏りを確認するときに使います。

---

## 主な機能

### 全クラスの一覧表示

- 学年とクラスの構成をもとに、表の行を自動で作ります
- 学年フィルターが指定されているときは、その学年だけを表示します
- 表示倍率は 100% を標準にして、見出し右側のボタンから拡大・縮小できます
- 左端のクラス見出し列は 100% 時の幅を保ったまま固定され、縮小した時も表全体は左寄せで表示されます
- 曜日ごとの最大時限数に合わせて列数を決め、使わない時限は無効セルとして表示します
- 各マスには教科名、担当教員名、必要なら B 週の教科を表示します
- TT 授業では、担当教員名の欄に参加教員を短い表記でまとめて表示します

### セルの選択

- マスをクリックすると、そのコマを選択します
- Ctrl / Cmd を押しながらクリックすると複数選択できます
- 選択した内容は右側の詳細パネルと連動します
- その曜日で存在しない時限は選択やドラッグの対象になりません

### ドラッグ&ドロップで入れ替え

- 授業が入っているマスは、別のマスへドラッグして入れ替えできます
- 入れ替え処理そのものは Zustand ストアの `swapTimetableEntries` が担当します

### スクロールしても見失いにくい固定見出し

- 上側の曜日・時限の見出しは、縦にスクロールしても上に固定されます
- 左側のクラス名は、横にスクロールしても左に固定されます
- 左上の「クラス」見出しは最前面に固定し、曜日ヘッダーより下に隠れないようにしています

---

## データの流れ

1. `useTimetableStore` から学年構成と時間割データを読み込みます
2. `rows` を作り、表の行として並べます
3. 各マスで `getEntry` を呼び、教科・教員・隔週情報を取り出します
4. クリックやドラッグ操作が行われると、親コンポーネントまたはストアに処理を渡します
