# PDF出力

> 対応ソースコード: `src/components/PdfExport.tsx`

## このファイルの役割

時間割をPDF形式で出力して、印刷したり、ファイルとして保存したりできるようにするコンポーネントです。学校で「この時間割表を紙に印刷して職員室に貼ろう」という時に使います。

---

## 主な機能

### PDF出力ボタン

- **何をするか**: 赤いボタン「📄 PDF出力」をクリックすると、出力の設定ダイアログが出現します
- **表示位置**: アプリの上部（ボタンバーの一部）
- **ソースコード**: 行396-410（ボタンの定義）

### 出力設定ダイアログ

クリックすると「何をPDF出力するか」を選ぶダイアログが出ます：

#### 出力オプション

1. **📅 時間割グリッド**
   - **内容**: 全クラスの時間割表（時間割の表（グリッド）と同じ）
   - **サイズ**: A4横版
   - **表示情報**:
     - クラス名
     - 曜日と時間
     - 各マスに教科と先生の名前
   - **ソースコード**: 行260-308（HTML生成部分）

2. **👩‍🏫 先生コマ数一覧**
   - **内容**: 先生ごとのスケジュール表（先生のスケジュール表と同じ）
   - **表示情報**:
     - 先生の名前と担当教科
     - 週のコマ数合計
     - 各先生が担当する全授業（何年何組の何教科か）
     - TT 授業は参加している先生全員の一覧に同じ授業が反映されます
   - **ソースコード**: 行310-385（HTML生成部分）

### 設定の選択

チェックボックスで両方、または片方だけを選べます：

```
✓ 📅 時間割グリッド
✓ 👩‍🏫 先生コマ数一覧

出力する ボタン
```

- **両方チェック**：2ページのPDFが出力（2ページ目は自動で改ページ）
- **片方だけ**：1ページのPDFが出力
- **何もチェックなし**：「出力する」ボタンが灰色になって押せなくなる
- **クリックしやすさ**: 各行はチェックボックスと見出しがひとまとまりになっていて、文字や余白の部分を押してもON/OFFできます

**ソースコード**: 行430-454（チェックボックスの描画）

## 出力内容の詳細

### 時間割グリッド（時間割表）

#### 表の構造

```
      月曜日      火曜日      水曜日      木曜日      金曜日
      1 2 3...  1 2 3...  1 2 3...  1 2 3...  1 2 3...
1-A  【】【】   【】【】   ...
1-B  【】【】   【】【】   ...
```

#### マスの内容

各マスには：
- **教科名**（太字、大きめ）
- **先生の名前**（小さめ、薄い色。TT の場合は参加教員をまとめて表示）
- 曜日ごとの最大時限数を超える場所は、PDF 上では「×」で対象外だと分かるように表示します

#### 隔週授業の表示

A週とB週が両方ある場合：
```
【A】国語
【B】社会
  田中
```

というように、両方を表示します。

**ソースコード**: 行285-307

### 先生ごとのコマ数（TeacherScheduleGrid形式）

#### 表の構造

```
先生名        週計  月  火  水  木  金
              計数  1 2 3... 1 2 3...
田中（国語）  18   【】【】 ...
佐藤（数学）  15   【】【】 ...
```

学校設定で「月曜は5限まで、木曜は6限まで」のように違いがある場合でも、PDF はその設定を反映して対象外のマスを分けて表示します。

#### マスの内容

色で区別されます：

| マスの色 | 内容 |
|---|---|
| 薄い青 | 単独担当 |
| 薄い紫 | B週隔週授業 |
| 薄い緑 | 先生グループ |
| 薄い黄 | 複数クラス合同 |

**ソースコード**: 行174-210（テーブルスタイルの定義）

## 出力後の操作

### 印刷ダイアログ

出力される新しいウィンドウに「🖨️ 印刷 / PDFとして保存」ボタンがあります。

#### PDFとして保存する場合

1. 「🖨️ 印刷」ボタンをクリック
2. ブラウザの印刷ダイアログが開く
3. **「送信先」で「PDFに保存」を選ぶ**
4. ファイル名を決めて保存

#### 紙に印刷する場合

1. 「🖨️ 印刷」ボタンをクリック
2. ブラウザの印刷ダイアログが開く
3. **「送信先」で使いたいプリンターを選ぶ**
4. 「印刷」をクリック

**ソースコード**: 行254-256（印刷ボタンの定義）

### ウィンドウを閉じる

「✕ 閉じる」ボタンで、出力ウィンドウを閉じます。

## HTML生成のポイント

このコンポーネントは、以下のように動作します：

1. **データ取得**: データストアから全ての時間割情報を取得
2. **HTML生成**: JavaScriptで手動でHTML文字列を組み立てる
3. **ウィンドウを開く**: `window.open()` で新しいウィンドウを開く
4. **HTMLを書き込む**: `document.write()` でHTML内容を書き込む
5. **印刷ダイアログを表示**: ユーザーが印刷またはPDF保存を選択

**ソースコード**: 行390-392（新規ウィンドウを開いてHTML を書き込み）

## スタイルの工夫

### 表のレイアウト

- **固定列幅**: 各セルの幅を固定して、きれいに配置
- **文字折り返し**：長い名前も自動で折り返す
- **中央揃え**: ほとんどのセルが中央に揃えられている

### 印刷とスクリーン用の分け方

CSS の `@media print` と `@media screen` で、以下を分けています：

| 状況 | 印刷用 | スクリーン用 |
|---|---|---|
| 背景色 | 白（余計な色を避ける） | グレー背景 |
| ボタン | 非表示 | 表示 |
| マージン | 0（ぴっちり） | あり（見やすく） |

**ソースコード**: 行215-249（CSSのメディアクエリ）
