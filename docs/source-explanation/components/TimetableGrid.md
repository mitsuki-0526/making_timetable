# 時間割の表（グリッド）

> 対応ソースコード: `src/components/TimetableGrid.tsx`

## このファイルの役割

このファイルは、時間割を大きな表で表示する部分です。学校の黒板に貼ってある時間割表をWebアプリで再現している、と想像してください。

**モニター上に表示される内容**
- 左側：クラスの名前（「1年A組」など）
- 上側：曜日（月火水木金）と時間帯（1限～6限）
- 中央：その時間に何の教科をどの先生が担当するか

このコンポーネントでは、表の表示だけでなく「操作」もできます。

---

## 主な機能

### 時間割表の表示

- **何をするか**: 学年ごとに、すべてのクラスと全時間帯の授業を表で表示します
- **表の見た目**: 
  - 普通のクラスは白背景
  - 特別支援学級は薄い黄色で区別
  - 1マスが1時間の1クラスの授業を表す
- **ソースコード**: `src/components/TimetableGrid.tsx`（クラス配置と表の描画）
- **関連ファイル**: [CellDropdown.md](#時間割の1マスセル)

### マスの選択（複数同時に選択）

- **何をするか**: Ctrl キー（Windows）または Cmd キー（Mac）を押しながらマスをクリックすると、複数のマスを同時に選ぶことができます
- **操作の注意**: Ctrl / Cmd を押したクリックは教科選択を開かず、そのマスを選択または選択解除します
- **見た目**: 選ばれたマスは青い枠で囲まれます
- **使い道**: 複数のマスを選んだ後、右クリックで「グループ化」できます。合同授業（複数クラスで一緒に授業）を作るときに使います
- **ソースコード**: 行51-76（Ctrl クリックの処理）
- **ESCキーで解除**: 選択を取り消したいときは、ESC キーを押します

### マスのドラッグ&ドロップ移動

- **何をするか**: マスをマウスで掴んで別の場所にドラッグ（引きずる）できます
- **どうなるか**: ドラッグしたマスの授業内容が、ドロップした先に入れ替わります
  - 例：「A組の月曜1限」を「A組の火曜1限」にドラッグすると、2つの授業が入れ替わります
- **できない場合**: 「固定コマ」に指定されている授業はドラッグできません（南京錠のマークが表示されます）
- **ソースコード**: 行153-177（ドラッグ&ドロップの処理）

### マスの見た目の工夫

- **マスが空の時**: 「未設定」と表示
- **マスに教科と先生が入っている時**: 教科名と先生の名前を表示
  - **隔週授業**の場合：A週とB週の両方を表示
  - **複数クラス合同**の場合：色付きの小さなマークで「🔗合同」と表示
- **警告の色分け**:
  - **黄色**：その日に同じ教科が複数回ある（注意が必要かも）
  - **赤色**：先生がまだ決まっていない

---

## データの流れ

1. ユーザーがマスをクリック → CellDropdown コンポーネントが起動
2. CellDropdown で教科や先生を選ぶ → 時間割表に反映
3. ユーザーがマスをドラッグ → TimetableGrid が交換処理を実行
4. マスの情報がリアルタイムで更新される
