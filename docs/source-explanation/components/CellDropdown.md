# 時間割の1マス（セル）

> 対応ソースコード: `src/components/CellDropdown.jsx`

## このファイルの役割

時間割表の1マス（1時間の1クラスの授業）をクリックすると現れるメニューやフォームを制御しています。このファイルが、ユーザーが「この時間に何の教科を配置するか」「どの先生に担当させるか」を決める場所を作っているわけです。

---

## 主な機能

### 左クリック：教科の選択

- **何をするか**: マスを左クリックするとドロップダウンメニューが現れて、そこから教科を選べます
- **クリックのしかた**: セルの文字の上でも下でも反応するように、セル全体がクリック対象になっています
- **表示方法**: 教科一覧は画面の手前に専用ポップアップとして表示し、背景の文字が重なって見えないようにしています
- **自動割り当て**: 教科を選ぶと、システムが自動で「その教科が教えられる先生」の中で最初に見つかった先生を割り当てます
  - 複数の先生が教えられる場合は、別のメニューが出て「どの先生にするか」と聞かれます
- **ソースコード**: `src/components/CellDropdown.jsx` 行79-109（教科の変更処理）
- **関連機能**: [store/useTimetableStore.md - setTimetableEntry](../store/useTimetableStore.md#settimetableentry---コマに教科と先生を配置する)

### 右クリック：詳細メニュー（隔週・グループ・先生変更）

右クリックすると、より細かい設定ができるメニューが現れます。

#### 担当を変更する

- **何をするか**: その教科を教えられるほかの先生に変更できます
- **使い道**: 例えば、最初に割り当てられた先生が忙しい場合、別の先生に切り替える
- **ソースコード**: 行217-232（右クリックメニューの「担当を変更」オプション）
- **関連機能**: [store/useTimetableStore.md - setTimetableTeacher](../store/useTimetableStore.md#settimetableteacher---先生だけを変更する)

#### 隔週設定（A週とB週）

- **何をするか**: 週によって違う教科を配置できます
  - 例：月曜1限は「A週は国語」「B週は社会」
- **背景**: 学校によっては「奇数週と偶数週で違う時間割を使う」ことがあります
- **表示**: マスに「A週」と「B週」が両方表示されたら隔週授業です
- **操作方法**:
  - 右クリック → 「隔週設定」を選ぶ
  - B週の教科を選ぶ
  - B週の先生も自動割り当てされます
- **ソースコード**: 行111-131（B週教科の変更）、行248-249（メニュー）
- **関連機能**: [store/useTimetableStore.md - setAltEntry](../store/useTimetableStore.md#setaltentry---隄週授業のb週を設定する)

#### グループ担当設定（複数クラスで一緒に授業）

- **何をするか**: 複数のクラスで一緒に授業をする場合「先生グループ」を使って管理できます
  - 例：「1年A組と1年B組が一緒に体育」の場合、体育の先生をグループとして登録
- **使い道**: 先生を1人ずつ管理するより、「このグループ」と指定するだけで複数クラスの先生管理が簡単になります
- **警告**: グループに含まれる先生に「配置不可の時間」が設定されている場合、警告メッセージが出ます
- **ソースコード**: 行132-152（グループ選択処理）、行250-266（メニューオプション）
- **関連機能**: [store/useTimetableStore.md](../store/useTimetableStore.md)

### 複数セルのグループ化

- **何をするか**: Ctrl クリックで複数のマスを選んだ後、右クリック → 「グループ化」を選ぶと、複数のマスが1つのグループとして扱われます
- **使い道**: 複数クラスの合同授業を一括管理する
- **見た目**: グループ化されたセルには色付きの枠が表示されます
- **ソースコード**: 行177-195（右クリックメニューのグループ化オプション）
- **関連機能**: [store/useTimetableStore.md - groupCells](../store/useTimetableStore.md#groupcells---セルをグループ化する)

### セルのグループ化を解除する

- **何をするか**: グループ化を取り消します
- **操作方法**: グループ内のマスで右クリック → 「グループ解除」を選ぶ
- **ソースコード**: 行196-213
- **関連機能**: [store/useTimetableStore.md - ungroupCells](../store/useTimetableStore.md#ungroupcells---グループを解除する)

### 教科をクリア（削除）

- **何をするか**: マスの内容を全て削除します
- **操作方法**: 右クリック → 「教科をクリア」
- **ソースコード**: 行268-283
- **関連機能**: [store/useTimetableStore.md - setTimetableEntry](../store/useTimetableStore.md#settimetableentry---コマに教科と先生を配置する)（null を渡す）

## 画面上に見える情報と警告

### マスの背景色

| 色 | 意味 |
|---|---|
| 白（無色）| 未設定のマス |
| 薄い青 | 選択中のマス |
| 薄い黄色 | 同じ日に同じ教科が複数ある（注意） |
| 薄い赤 | 先生がまだ決まっていない |

### マスに表示される小さなマーク

| マーク | 意味 |
|---|---|
| 🔒 | 固定コマ（動かせません） |
| ⠿ | ドラッグできる授業内容がある |
| 🔗合同 | 複数クラスで一緒に授業 |
| 隔週 | A週とB週で異なる授業 |

### 警告ダイアログ

グループ担当を設定した先生に「配置不可の時間」が含まれていると、黄色い警告ダイアログが出ます。内容を確認してから「確認しました」を押します。

---

## データの流れ

1. ユーザーがマスをクリック → セルの内容が表示
2. 左クリック（教科選択）または右クリック（詳細メニュー）
3. メニューから操作を選択 → ポップアップフォームが出現
4. フォームで選択 → データストア（useTimetableStore）に保存
5. 保存されたデータが時間割表に反映
