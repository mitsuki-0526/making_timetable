# TT担当ヘルパー

> 対応ソースコード: `src/lib/teamTeaching.ts`

## このファイルの役割

1つの授業に複数の先生が同時に入る TT（チームティーチング）を、アプリ全体で同じ考え方で扱うための共通部品です。

以前は「先生グループ」という名前だけを持っていましたが、このファイルでは「この授業には誰が参加しているか」を先生 ID の配列としてまとめ直します。これにより、表示・保存・チェックが同じ情報を見るようになりました。

---

## 主な機能

### 参加教員一覧を取り出す

- 通常授業なら1人、TT授業なら複数人という形で、そのコマに参加している先生の一覧を返します
- A週とB週で別々の担当にも対応しています

### 表示用の先生名を作る

- セルやPDFに表示するために、参加している先生名を読みやすい文字列へ変換します
- 省略表示と通常表示の両方に対応しています

### 保存データを整える

- `teacher_group_id` があるコマは、保存前や読み込み時に `teacher_ids` / `alt_teacher_ids` を補って、全員参加の情報が欠けないようにします

### 「この先生が参加しているか」を判定する

- 先生別時間割やエラーチェックで、個人担当でも TT でも同じ関数で判定できます
