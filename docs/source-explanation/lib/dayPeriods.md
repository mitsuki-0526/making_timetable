# 曜日別時限ヘルパー

> 対応ソースコード: `src/lib/dayPeriods.ts`

## このファイルの役割

このファイルは、「月曜は5限まで、木曜は6限まで」のような学校ごとの違いを、アプリ全体で同じ基準で扱うための共通ヘルパーです。

以前は多くの画面やロジックが「毎日1〜6限ある」前提で動いていましたが、このファイルを通すことで、表示・出力・検証・自動生成が同じ設定を見るようになりました。

---

## 主な機能

### 既定値をまとめる

- 何も設定されていない場合は、月〜金をすべて6限までとして扱います
- 古い保存データを読み込んだ時も、この既定値で不足分を補います

### 曜日ごとの使用可能時限を返す

- ある曜日で何限まで使えるかを返します
- たとえば「月曜は5限まで」なら、月曜で使える一覧は「1,2,3,4,5」になります

### 画面に出す時限の上限を決める

- 全曜日の中で一番遅い時限を探して、表の行数や列数を決めます
- これにより、どこかの曜日だけ6限がある学校でも、表全体の形を崩さずに表示できます

### そのコマが有効か判定する

- 指定した曜日・時限が、その学校設定で使えるコマかどうかを真偽値で返します
- 各画面はこの結果を見て、クリックできるセルか無効セルかを切り替えます
