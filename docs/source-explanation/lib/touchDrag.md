# タッチD&Dヘルパー

> 対応ソースコード: `src/lib/touchDrag.ts`

## このファイルの役割

このファイルは、スマートフォンやタブレットのようにマウスのドラッグ機能がそのまま使えない環境で、左パレットから時間割セルへ項目を運べるようにするための共通ヘルパーです。

ブラウザの標準的なドラッグ&ドロップはタッチ操作に弱いため、このファイルでは「青い小さな札」を一時的に画面へ出し、指の動きに合わせて追従させています。

---

## 主な機能

### ドラッグ開始を記録する

- いま運んでいるのが教科なのか、先生なのか、TT設定なのかを覚えます
- その情報に合わせて、青いラベルを画面に表示します

### 指の動きに合わせて追従する

- タッチ移動のたびに、青いラベルの位置を更新します
- これにより、ユーザーは「今これを持っている」と視覚的に確認できます

### 指を離した場所のセルを調べる

- タッチ終了時に、その位置の下にある時間割セルを探します
- 見つかったセルが対象なら、対応する教科・先生・TT設定をストアへ反映します

### 中断時もラベルを片付ける

- タッチ操作は、端末やブラウザの都合で途中キャンセルされることがあります
- その場合でも、画面上に青いラベルだけが残らないよう、終了時と同じ後始末を共通で呼ぶようにしています
