# 自動作成ソルバー

> 対応ソースコード: `src/lib/jsSolver.worker.ts`

## このファイルの役割

時間割を自動で組み立てる中核です。先生の勤務不可、同時刻の重複、合同授業、隔週授業、TT などを見ながら、ブラウザの中で候補を何度も試して一番よい案を選びます。

このファイルは「とにかく全部埋める」だけではなく、まず危険な違反を減らすことを優先します。特に、先生が同じ時間に複数クラスへ入ってしまう状態と、勤務不可の時間に授業を入れてしまう状態は hard 違反として先に解消する考え方です。

---

## 主な機能

### 時間割案を自動生成する

- 空いているコマへ教科を入れ、担当できる先生を探して配置します
- 1回だけで決めず、時間制限の中で複数回試して、より条件のよい案を選びます
- 曜日ごとの最大時限数の設定がある場合は、その曜日で存在しない時限には最初から配置しません

### 危険な違反を優先して修復する

- 先生の同時刻重複や勤務不可違反が残った場合は、教員の再割当やコマ移動を試します
- それでも直らない時は、deep repair で問題のコマを重点的に組み直します

### 同じ曜日・同じ学年の中で入れ替える

- deep repair では、同じ曜日・同じ学年の別クラス・別時限どうしで 2 コマを入れ替える修復も行います
- この時は A週教科だけでなく B週教科もセットで動かし、TT の参加教員も含めて担当を組み直します
- hard 違反が実際に減る手だけ採用するため、見た目だけ変えて悪化する入れ替えは残しません
