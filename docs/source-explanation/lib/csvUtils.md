# CSV / Excel 入出力ヘルパー

> 対応ソースコード: `src/lib/csvUtils.ts`

## このファイルの役割

このファイルは、基礎構成の一覧を Excel に書き出したり、Excel から読み戻したりするための共通ヘルパーです。

画面側は「書き出したい」「読み込みたい」とお願いするだけにして、実際の表の形や保存形式の細かい扱いはこのファイルにまとめています。

---

## 主な機能

### 基礎構成を Excel に変換する

- クラス編成と教員リストを 1 つの Excel ファイルにまとめます
- クラス編成シートでは、学年・クラス・特支区分・教科・時数を 1 行ずつ並べます
- 教員リストシートでは、先生名、担当教科、対象学年、配置不可時間を保存します

### Excel から基礎構成を読み戻す

- 書き出した Excel を読み取り、クラス編成と教員リストをアプリのデータ形式へ戻します
- 古い保存データや空欄があっても、読める範囲で安全に取り込みます

### ブラウザ版とデスクトップ版で保存方法を切り替える

- ブラウザ版では通常のダウンロード機能を使います
- Tauri のデスクトップ版では保存ダイアログで選んだ場所へ直接書き込みます
- 現在の Tauri 版は `plugin-fs` の `writeFile(path, data)` 形式で保存し、保存先パスが文字列として正しく渡るようにしています
