Official Documentation

時間割作成アシストツール

AIの直感的な翻訳力と、数理最適化の計算エンジン。学校現場に最高の生産性を届けるデスクアプリケーション・マニュアル

01 | クイックスタートガイド

製品を使いこなすための3つのコアプロセスを紹介します。

Step 01

条件の入力

先生方がUI（チェックボックス等）から、出勤不可時間や規定時数などのベース条件を詳細に入力します。

Step 02

自動配置

計算エンジンが1～3分の演算を行い、80%～95%程度の完成度を持つドラフト時間割を生成します。

Step 03

手動微調整

自動生成された結果に対し、ドラッグ＆ドロップで最後の人間にしかできない「感性」の調整を行います。

✨

高度なAI相談機能

手動入力では説明しにくい「特別な配慮」が必要な条件は、AIにチャットで相談できます。
※本機能は開発者モードでの詳細設定が必要です。

02 | 実装済み機能の詳細

⌨️ 高度な操作

複数セル選択、右クリックでの合同コマ一括設定、隔週授業対応。

🔗 支援学級連動

通常学級の変更に合わせ、支援学級側の授業もあらかじめ決めたルールで自動差し替え。

⚠️ 即時検証

規定時数の過不足、教員の重複、集中配置などをリアルタイムで警告。ミスを視覚化。

💾 データ管理

JSON形式での読み書き、File System API による上書き保存（対応ブラウザのみ）。

03 | セキュリティと構成

完全ローカル、完全独立

本システムは AI本体（Gemma 4）を内部に同梱。外部APIへの通信を行わず、校内の機密情報を守り抜きます。

レイヤー 役割
UIReact / Electron による直感的なインターフェース
EnginePython (FastAPI) による高速演算
OptimizerGoogle OR-Tools による数理最適化
